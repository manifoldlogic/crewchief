# ⚠️ DEPRECATED: maproom-mcp

**This package is no longer maintained and will not receive updates.**

## Use the New Package Instead

This package has been moved to a scoped package:

### Install the new package:
```bash
npm install @crewchief/maproom-mcp
```

or with npx:
```bash
npx @crewchief/maproom-mcp setup --provider=openai
```

## What Changed?

- **Old package**: `maproom-mcp` (deprecated, no longer updated)

The functionality is identical, only the package name has changed.

## Migration

Simply replace `maproom-mcp` with `@crewchief/maproom-mcp` in your commands and configurations:

**Before:**
```bash

```

**After:**
