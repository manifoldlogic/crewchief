These files contain definitions for specialized AI agents. They are high-level definitions, not specifically in the Claude Code subagent format. They are used to guide the creation of subagents, not as subagents themselves. When creating a file in this directory, focus on what the agent should be focused on. First check if there is already a different agent in this folder or in .claude/agents that covers the same need before creating a new agent. Only create the agent if it could perform it's role better as a specialized agent than a general agent.
