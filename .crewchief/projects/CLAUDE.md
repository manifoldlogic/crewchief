# .crewchief/projects — Active Projects

## Naming Conventions

- **Project folders**: `{SLUG}_{descriptive-name}` (e.g., `DKRHUB_docker-hub-publishing`)
- **Tickets**: `{SLUG}-{NUMBER}_{description}.md` (e.g., `DKRHUB-1001_setup.md`)
- **Ticket numbers**: Start at 1001 for Phase 1, 2001 for Phase 2
- **Planning docs**: `analysis.md`, `architecture.md`, `plan.md`, `quality-strategy.md`

## Completion Criteria

A project is complete when:
- All tickets have `- [x] **Verified**` checkbox
- No active development planned
- Knowledge synthesized to `/docs/` (if applicable)

Complete projects move to `../archive/projects/`.
