## Overview

This directory contains documentation from genetic optimization experiments that tested different tool description variants to improve AI agent performance on code search benchmarks.

**Key Findings**:
- Baseline (variant-control): 17.7%
- Winner (variant-a-detailed): 19.6% (+1.9%)
- Transformation workflows significantly outperform static examples
- Performance plateaued at 19-20% across Gen 2-10

See detailed results and patterns in the linked documents above.
