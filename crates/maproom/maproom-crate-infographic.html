maproom

Semantic Code Search Engine for the Rust Ecosystem

A powerful binary for semantic code search that indexes your codebase using tree-sitter, stores chunks in SQLite with FTS5 and vector extensions, enabling both full-text and semantic similarity search.

Core Architecture
🌳
Tree-sitter Parsing
Parse code into abstract syntax trees with support for 10+ languages for precise code understanding.
🔍
Dual Search
Full-text search with FTS5 and semantic similarity search using vector embeddings.
⚡
SQLite Storage
Lightweight, embedded database with FTS5 for full-text and sqlite-vec for vector queries.
🤖
Multiple Embeddings
Ollama (free), OpenAI, or Google Vertex AI with automatic provider detection.

Environment Variables
MAPROOM_EMBEDDING_BATCH_SIZE
Batch size for embedding generation (default: 100)
MAPROOM_EMBEDDING_CACHE_SIZE
In-memory embedding cache capacity (default: 10000)
MAPROOM_EMBEDDING_CACHE_TTL
Cache entry time-to-live in seconds (default: 3600)
MAPROOM_EMBEDDING_DIMENSION
Embedding vector dimension (auto-detected from provider)
OPENAI_API_KEY
OpenAI API key for embedding generation
GOOGLE_APPLICATION_CREDENTIALS
Path to Google Cloud service account JSON
