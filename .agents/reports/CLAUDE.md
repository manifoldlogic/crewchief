Files and folders in the .agents/reports directory are reports generated by the AI agents.

## Directory Structure and Naming Conventions

Folders and files in the root of the reports directory should be named starting with a date in the format YYYY-MM-DD.

```
.agents/reports/
├── {DATE}_report_name      // report folder name
│   │── analysis.md         // example report file in a report folder
│   └── ...
├── {DATE}_report_name.md   // example single report file in the root of the reports directory
└── ...
```
